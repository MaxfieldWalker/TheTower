# 3Dサウンド

AudioListenerとAudioSourceという概念があり，
デフォルトでAudioListenerはカメラに設定されている。
AudioSourceにはCubeなどのGameObjectを設定できる。

3Dサウンドに関するプロパティが最初からあるので，
プログラミングなしで3Dサウンドを実現できる。
